#### Graphite Lattice Visualization

Constructed a 3D ABA-stacked graphite lattice using lattice vectors and basis atoms in Python.

Features:
- Hexagonal lattice generation
- ABA stacking of graphene layers
- Nearest-neighbor bond visualization
- Interlayer spacing visualization
- 3D crystal structure rendering with Matplotlib

Tools:
- Python
- NumPy
- Matplotlib

This project was completed as part of an undergraduate Computational Physics course.
